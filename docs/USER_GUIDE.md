## 1. เริ่มต้นใช้งาน

### 1.1 การเข้าสู่ระบบ (Login)

1. เปิด Browser (แนะนำ Google Chrome)
2. เข้า URL ของระบบ: `https://your-school.scidas.app`
3. กรอก **Email** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแลระบบ
4. คลิกปุ่ม **"เข้าสู่ระบบ"**

> **หมายเหตุ:** หากลืมรหัสผ่าน คลิก "ลืมรหัสผ่าน" เพื่อรีเซ็ตผ่าน Email

### 1.2 หน้าจอหลัก

เมื่อ Login สำเร็จ ระบบจะแสดง Dashboard ตามบทบาทของผู้ใช้:

| บทบาท | หน้าหลัก |
|-------|---------|
| Admin | Dashboard ภาพรวม + เมนูจัดการผู้ใช้ |
| ผู้บริหาร | Dashboard ภาพรวมทั้งโรงเรียน |
| ครูประจำชั้น | Dashboard ห้องเรียนของตัวเอง |
| ครูแนะแนว | Dashboard เคสความเสี่ยงและการช่วยเหลือ |
| ครูผู้สอน | หน้าบันทึกคะแนน/พฤติกรรม |

### 1.3 โครงสร้างเมนู

## 2. หน้า Dashboard

### 2.1 Dashboard ผู้บริหาร / Admin

แสดงภาพรวมทั้งโรงเรียน:

**Stat Cards (ด้านบน):**
- 🧑‍🎓 จำนวนนักเรียนทั้งหมด
- 🟢 นักเรียนกลุ่มปกติ
- 🟡 นักเรียนกลุ่มเฝ้าระวัง
- 🔴 นักเรียนกลุ่มเสี่ยงสูง

**Charts (กลาง):**
- 📊 Pie Chart: สัดส่วนระดับความเสี่ยง
- 📊 Bar Chart: เปรียบเทียบรายชั้น/ห้อง
- 📊 Line Chart: แนวโน้มการมาเรียนรายเดือน

**Tables (ด้านล่าง):**
- ⚠️ Top 10 นักเรียนเสี่ยงสูง
- 📅 นักเรียนขาดเรียนบ่อย
- 🤝 เคสรอดำเนินการ

**Filters (ตัวกรอง):**
- ปีการศึกษา
- ภาคเรียน
- ระดับชั้น

### 2.2 Dashboard ครูประจำชั้น

แสดงข้อมูลเฉพาะห้องที่รับผิดชอบ:
- สถิติการมาเรียนวันนี้
- รายชื่อนักเรียนที่ขาดเรียนวันนี้
- นักเรียนกลุ่มเสี่ยงในห้อง
- เคสที่ต้องติดตาม
- แผน IDP ที่กำลังดำเนินการ

## 3. จัดการข้อมูลนักเรียน

### 3.1 ดูรายชื่อนักเรียน

1. คลิกเมนู **"นักเรียน"** ด้านซ้าย
2. ระบบแสดงตารางรายชื่อนักเรียน
3. ใช้ช่องค้นหาเพื่อหาชื่อ/เลขประจำตัว
4. ใช้ Dropdown กรองตามชั้น/ห้อง
5. คลิกที่ชื่อนักเรียนเพื่อดู Profile

### 3.2 เพิ่มนักเรียนใหม่

1. คลิกเมนู **"นักเรียน"** → คลิกปุ่ม **"+ เพิ่มนักเรียน"**
2. กรอกข้อมูลตามแบบฟอร์ม:

**ข้อมูลทั่วไป:**
| ช่อง | จำเป็น | คำอธิบาย |
|------|--------|---------|
| เลขประจำตัว | ✅ | เลขประจำตัวนักเรียน |
| คำนำหน้า | ✅ | เด็กชาย/เด็กหญิง/นาย/นางสาว |
| ชื่อ | ✅ | ชื่อจริง |
| นามสกุล | ✅ | นามสกุล |
| วันเกิด | ✅ | วัน/เดือน/ปี เกิด |
| เพศ | ✅ | ชาย/หญิง |
| ชั้น/ห้อง | ✅ | เช่น ป.1/1 |
| โรคประจำตัว | ❌ | ระบุถ้ามี |
| วิธีเดินทาง | ✅ | เดินเท้า/จักรยาน/รถจักรยานยนต์/รถยนต์/รถโรงเรียน/อื่นๆ |

**ข้อมูลผู้ปกครอง:**
| ช่อง | จำเป็น | คำอธิบาย |
|------|--------|---------|
| ชื่อผู้ปกครอง | ✅ | ชื่อ-นามสกุล |
| ความสัมพันธ์ | ✅ | บิดา/มารดา/ปู่/ย่า/ตา/ยาย/อื่นๆ |
| เบอร์โทร | ✅ | เบอร์โทรศัพท์ |
| อาชีพ | ❌ | อาชีพผู้ปกครอง |

**ที่อยู่:**
| ช่อง | จำเป็น |
|------|--------|
| บ้านเลขที่ | ✅ |
| หมู่ที่ | ❌ |
| ตำบล | ✅ |
| อำเภอ | ✅ |
| จังหวัด | ✅ |
| รหัสไปรษณีย์ | ❌ |

**สถานะครอบครัว:**
| ช่อง | ตัวเลือก |
|------|---------|
| สถานะครอบครัว | อยู่พร้อมหน้า / บิดามารดาแยกกัน / บิดาเสียชีวิต / มารดาเสียชีวิต / กำพร้า / อื่นๆ |
| ภาวะเศรษฐกิจ | ดี / ปานกลาง / ยากจน / ยากจนพิเศษ |

3. คลิก **"บันทึก"**

### 3.3 นำเข้าจาก Excel

1. คลิก **"นักเรียน"** → **"นำเข้าข้อมูล"**
2. ดาวน์โหลด Template Excel
3. กรอกข้อมูลตาม Template
4. อัปโหลดไฟล์ (รองรับ .xlsx, .csv)
5. ตรวจสอบ Preview ข้อมูล
6. แก้ไข Error (ถ้ามี)
7. คลิก **"นำเข้า"**

### 3.4 หน้า Profile นักเรียน

หน้า Profile แสดงข้อมูลนักเรียนแบบ 360° แบ่งเป็น Tabs:

| Tab | เนื้อหา |
|-----|---------|
| **ข้อมูลทั่วไป** | ข้อมูลพื้นฐาน ผู้ปกครอง ที่อยู่ ครอบครัว |
| **การมาเรียน** | สถิติ + กราฟการมาเรียน |
| **ผลการเรียน** | คะแนน เกรด ทักษะพื้นฐาน แนวโน้ม |
| **พฤติกรรม** | ประวัติพฤติกรรม การส่งงาน |
| **เยี่ยมบ้าน** | ประวัติเยี่ยมบ้าน รูปภาพ |
| **การช่วยเหลือ** | ประวัติการรับความช่วยเหลือ |
| **แผนพัฒนา** | แผน IDP ทั้งหมด |

ด้านบนของ Profile แสดง:
- 🟢/🟡/🔴 **Risk Badge** ตามระดับความเสี่ยง
- 📊 **Risk Score** คะแนนความเสี่ยง
- 📅 **อัตราการมาเรียน** เปอร์เซ็นต์

## 4. บันทึกการมาเรียน

### 4.1 บันทึกรายวัน

1. คลิก **"การมาเรียน"** → **"บันทึก"**
2. เลือก **วันที่** (ค่าเริ่มต้นคือวันนี้)
3. เลือก **ชั้น/ห้อง**
4. ระบบแสดงรายชื่อนักเรียนในห้อง
5. เลือกสถานะสำหรับแต่ละคน:

| สถานะ | สี | ไอคอน |
|-------|-----|------|
| มาเรียน | 🟢 เขียว | ✅ |
| ขาดเรียน | 🔴 แดง | ❌ |
| ลาป่วย | 🟡 เหลือง | 🤒 |
| ลากิจ | 🟠 ส้ม | 📋 |
| มาสาย | 🟣 ม่วง | ⏰ |

6. เพิ่มหมายเหตุ (ถ้ามี)
7. คลิก **"บันทึก"**

> **Tips:** สามารถคลิก "มาทุกคน" เพื่อตั้งค่าเริ่มต้นเป็น "มาเรียน" ทุกคน แล้วแก้เฉพาะคนที่ไม่มา

### 4.2 ดูสถิติการมาเรียน

1. คลิก **"การมาเรียน"** (หน้า Overview)
2. เลือกช่วงเวลา: รายวัน / รายสัปดาห์ / รายเดือน / รายเทอม
3. ดูกราฟและตารางสถิติ

**ข้อมูลที่แสดง:**
- เปอร์เซ็นต์การมาเรียนภาพรวม
- จำนวนมา/ขาด/ลา/สาย
- นักเรียนที่การมาเรียนต่ำกว่า 80% (Highlight สีแดง)
- กราฟเทียบรายเดือน

## 5. บันทึกผลการเรียน

### 5.1 บันทึกคะแนน

1. คลิก **"ผลการเรียน"** → **"บันทึกคะแนน"**
2. เลือก **วิชา** และ **ชั้น/ห้อง**
3. เลือก **ประเภท**: คะแนนสอบ / คะแนนงาน / คะแนนรวม
4. กรอกคะแนนเต็มและคะแนนที่ได้
5. คลิก **"บันทึก"**

### 5.2 บันทึกทักษะพื้นฐาน

1. คลิก **"ผลการเรียน"** → **"ทักษะพื้นฐาน"**
2. เลือก **ชั้น/ห้อง** และ **ภาคเรียน**
3. ประเมินทักษะสำหรับแต่ละนักเรียน:

| ทักษะ | ระดับ |
|-------|-------|
| การอ่าน | ดีมาก / ดี / พอใช้ / ปรับปรุง / ไม่ผ่าน |
| การเขียน | ดีมาก / ดี / พอใช้ / ปรับปรุง / ไม่ผ่าน |
| การคิดเลข | ดีมาก / ดี / พอใช้ / ปรับปรุง / ไม่ผ่าน |

4. เพิ่มหมายเหตุ (ถ้ามี)
5. คลิก **"บันทึก"**

---

## 6. บันทึกพฤติกรรมและการส่งงาน

### 6.1 บันทึกพฤติกรรม

1. คลิก **"พฤติกรรม"** → **"บันทึก"**
2. เลือก **นักเรียน** (ค้นหาด้วยชื่อ)
3. เลือก **ประเภท**: พฤติกรรมดี / พฤติกรรมที่ควรติดตาม
4. เลือก **หมวดหมู่**: การเรียน / สังคม / อารมณ์ / ระเบียบวินัย
5. เลือก **ระดับความรุนแรง** (สำหรับพฤติกรรมที่ควรติดตาม): ปกติ / น่าเป็นห่วง / ร้ายแรง
6. อธิบายพฤติกรรม
7. คลิก **"บันทึก"**

### 6.2 บันทึกการส่งงาน

1. คลิก **"พฤติกรรม"** → **"การส่งงาน"**
2. เลือก **วิชา** และ **ชั้น/ห้อง**
3. ระบุ **ชื่องาน** และ **กำหนดส่ง**
4. เลือกสถานะ:

| สถานะ | สี |
|-------|-----|
| ส่งแล้ว | 🟢 เขียว |
| ส่งช้า | 🟡 เหลือง |
| ไม่ส่ง | 🔴 แดง |

5. คลิก **"บันทึก"**

---

## 7. เยี่ยมบ้าน

### 7.1 บันทึกการเยี่ยมบ้าน

1. คลิก **"เยี่ยมบ้าน"** → **"บันทึกใหม่"**
2. เลือก **นักเรียน**
3. กรอกข้อมูล:

| ช่อง | คำอธิบาย |
|------|---------|
| วันที่เยี่ยม | วัน/เดือน/ปี |
| สภาพบ้าน | สภาพแวดล้อมทั่วไป |
| ผู้ที่พบ | เช่น มารดา, ย่า |
| ปัญหาที่พบ | เช่น ฐานะยากจน, ขาดผู้ดูแล |
| ความต้องการ | เช่น ทุนการศึกษา, อุปกรณ์ |
| ข้อเสนอแนะ | คำแนะนำจากครู |

4. อัปโหลดรูปภาพ (สูงสุด 5 รูป)
5. คลิก **"บันทึก"**

## 8. การช่วยเหลือนักเรียน

### 8.1 สร้างเคสช่วยเหลือ

1. คลิก **"การช่วยเหลือ"** → **"สร้างเคสใหม่"**
2. เลือก **นักเรียน**
3. เลือก **ประเภทการช่วยเหลือ**:
   - ทุนการศึกษา
   - อุปกรณ์การเรียน
   - อาหารกลางวัน
   - เครื่องแต่งกาย
   - คำปรึกษา/ให้กำลังใจ
   - ส่งต่อหน่วยงานภายนอก
   - อื่นๆ
4. กรอกรายละเอียด
5. ระบุผู้รับผิดชอบ
6. คลิก **"บันทึก"**

### 8.2 อัปเดตสถานะ

1. คลิกที่เคสในรายการ
2. เปลี่ยนสถานะ:
   - 🔵 **รอดำเนินการ** → เพิ่งสร้าง
   - 🟡 **กำลังดำเนินการ** → อยู่ระหว่างช่วยเหลือ
   - 🟢 **สำเร็จ** → ช่วยเหลือเสร็จแล้ว
   - ⚫ **ยกเลิก** → ยกเลิกเคส
3. เพิ่มการติดตามผล (Follow-up)
4. คลิก **"บันทึก"**

## 9. วิเคราะห์ความเสี่ยง (Early Warning)

### 9.1 ดูภาพรวมความเสี่ยง

1. คลิก **"วิเคราะห์ความเสี่ยง"**
2. ระบบแสดง:
   - **Stat Cards**: จำนวนนักเรียนแต่ละระดับ
   - **Pie Chart**: สัดส่วนระดับความเสี่ยง
   - **Table**: รายชื่อนักเรียนเรียงตาม Risk Score

### 9.2 ดูรายละเอียดความเสี่ยงรายบุคคล

1. คลิกชื่อนักเรียนในตาราง
2. ระบบแสดง:
   - **Risk Score** ปัจจุบัน
   - **Risk Level** (ปกติ/เฝ้าระวัง/เสี่ยงสูง)
   - **ปัจจัยเสี่ยง** ที่ตรวจพบ พร้อมคะแนน
   - **Risk Trend** ย้อนหลัง

### 9.3 สัญลักษณ์ระดับความเสี่ยง

| ระดับ | สี | Score | ความหมาย |
|-------|-----|-------|---------|
| 🟢 ปกติ | เขียว | 0-30 | ไม่มีความเสี่ยง ดูแลปกติ |
| 🟡 เฝ้าระวัง | เหลือง | 31-60 | เริ่มมีสัญญาณ ต้องติดตามใกล้ชิด |
| 🔴 เสี่ยงสูง | แดง | 61-100 | ต้องดำเนินการช่วยเหลือทันที |

### 9.4 Flag นักเรียนแบบ Manual

ครูสามารถ Flag นักเรียนที่เห็นว่าควรติดตาม แม้ Risk Score จะยังไม่สูง:

1. ไปที่หน้า Profile นักเรียน
2. คลิก **"🚩 Flag สำหรับติดตาม"**
3. เพิ่มเหตุผล
4. คลิก **"บันทึก"**

## 10. แผนพัฒนารายบุคคล (IDP)

### 10.1 สร้างแผนพัฒนาใหม่

1. คลิก **"แผนพัฒนา"** → **"สร้างแผนใหม่"**
2. เลือก **นักเรียน**
3. ระบบจะแสดง **จุดที่ควรพัฒนา** อัตโนมัติ (จาก Risk Factors)
4. กรอกข้อมูลแผน:

| ช่อง | คำอธิบาย |
|------|---------|
| ชื่อแผน | เช่น "แผนพัฒนาการอ่าน ด.ช.สมชาย" |
| ปัญหา/สาเหตุ | อธิบายปัญหาที่พบ |
| วันเริ่มต้น | วันที่เริ่มแผน |
| วันสิ้นสุด | วันที่คาดว่าจะเสร็จ |

5. เพิ่ม **เป้าหมาย** (Goals):
   - เช่น "มาเรียนไม่น้อยกว่า 90%"
   - เช่น "อ่านหนังสือออกเสียงได้คล่องขึ้น"
   - เช่น "ส่งงานครบทุกวิชา"

6. เพิ่ม **กิจกรรมช่วยเหลือ** (Activities):
   - เช่น "ฝึกอ่านวันละ 15 นาที"
   - เช่น "ประสานผู้ปกครองรายสัปดาห์"
   - เช่น "เรียนเสริมหลังเลิกเรียน"

7. คลิก **"บันทึก"**

### 10.2 ติดตามผลแผน

1. คลิกที่แผนในรายการ
2. เพิ่มการประเมินผล:

| ช่อง | คำอธิบาย |
|------|---------|
| วันที่ประเมิน | วัน/เดือน/ปี |
| ผลก่อนช่วยเหลือ | คะแนน/ระดับก่อน |
| ผลหลังช่วยเหลือ | คะแนน/ระดับหลัง |
| สรุปผล | ดีขึ้น / เท่าเดิม / แย่ลง |
| หมายเหตุ | รายละเอียดเพิ่มเติม |

3. อัปเดตสถานะแผน:
   - 📝 **ร่าง** → กำลังวางแผน
   - 🔄 **ดำเนินการ** → กำลังทำตามแผน
   - ✅ **สำเร็จ** → บรรลุเป้าหมาย
   - ❌ **ยกเลิก** → ยกเลิกแผน

## 13. คำถามที่พบบ่อย

### Q: ลืมรหัสผ่าน ทำอย่างไร?
**A:** คลิก "ลืมรหัสผ่าน" ที่หน้า Login → กรอก Email → ระบบจะส่งลิงก์รีเซ็ตรหัสผ่านไปที่ Email

### Q: ทำไม Risk Score ไม่อัปเดต?
**A:** Risk Score จะคำนวณใหม่อัตโนมัติเมื่อมีการอัปเดตข้อมูลที่เกี่ยวข้อง เช่น การมาเรียน คะแนน พฤติกรรม หากต้องการคำนวณใหม่ทั้งหมด ให้ติดต่อ Admin

### Q: สามารถใช้บนมือถือได้ไหม?
**A:** ได้ ระบบออกแบบให้ Responsive ใช้งานได้บนมือถือ แท็บเล็ต และคอมพิวเตอร์

### Q: ข้อมูลนักเรียนปลอดภัยไหม?
**A:** ข้อมูลถูกเข้ารหัส (HTTPS) และมีระบบ Row Level Security ที่ทำให้ผู้ใช้เห็นเฉพาะข้อมูลที่มีสิทธิ์เข้าถึงเท่านั้น

### Q: สามารถ Export ข้อมูลได้ไหม?
**A:** ได้ ทุกรายงานสามารถ Export เป็น PDF หรือ Excel ได้

### Q: ระบบรองรับนักเรียนกี่คน?
**A:** ระบบออกแบบสำหรับโรงเรียนขนาดเล็ก (< 120 คน) แต่สามารถรองรับได้มากกว่าหากจำเป็น
